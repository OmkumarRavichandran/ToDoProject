# Title : TODO Application


# Description

TODO Application :

ToDo List App is a kind of app that generally used to maintain our day-to-day tasks or list everything that we have to do. 
It is helpful in planning our daily schedules.

Features of TODO Application :
To-do lists offer a way to increase productivity, stopping you from forgetting things, helps prioritise tasks, 
manage tasks effectively, use time wisely and improve time management as well as workflow.
.

# General Information

1.This TODO application is used for creating task in a Digital perspective.
2.Order Tasks – This one is really simple. ...
3.Prioritization – This is another basic time management feature. ...
4.Simple UI – Following on from quick capture, overall the app should be simple.
